# はじめに
<!-- position: 1 -->

Bluditは、あなたの**Webサイト**や**blog**をほんの数秒で構築できるWebアプリケーションです。そして、完全に**自由(フリー)でオープンソース**です。Bluditは、フラットファイルCMSです。(この場合)Bluditはコンテンツを**JSON**形式で保存することを意味しています。データベースのインストールや構築の必要はありません。必要なものはPHPが利用できるWebサーバーだけです。
